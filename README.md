# TryHackMe Write-Ups

Welcome to my collection of TryHackMe write-ups.

This repository contains detailed walkthroughs of rooms I've completed on TryHackMe. The goal is to document my learning process, share different approaches to solving challenges, and build a personal knowledge base for future reference.

## What's Included

* Step-by-step room solutions
* Enumeration techniques
* Privilege escalation methods
* Notes on tools and commands used
* Lessons learned from each room

## Disclaimer

These write-ups are intended for educational purposes only. All challenges were completed within the legal and authorized environment provided by TryHackMe.

## Why This Repository?

Cybersecurity is all about continuous learning. Writing down the process helps reinforce concepts, improve methodology, and track progress over time. Hopefully, these notes can also help other learners who are working through similar rooms.

## Structure

Each room has its own directory containing:

* Room overview
* Enumeration phase
* Exploitation steps
* Privilege escalation (if applicable)
* Final thoughts and takeaways

## Tools Commonly Used

* Nmap
* Gobuster
* Burp Suite
* Metasploit
* LinPEAS
* Wireshark
* Hydra
